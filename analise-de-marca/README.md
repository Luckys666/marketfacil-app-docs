# Análise de Marca

A **Análise de Marca** consulta a base oficial do INPI (Instituto Nacional da Propriedade Industrial) e mostra se o nome de uma marca está registrado. Essencial antes de usar qualquer marca em anúncios — marca registrada pode derrubar seu anúncio via **BPP** (Brand Protection Program) do Mercado Livre e até bloquear sua conta.

## Como usar

1. No menu lateral, clique em **Análise de Marca**.
2. Digite o nome da marca (ex: "Nike").
3. Clique em **Analisar**.
4. Aguarde alguns segundos e veja o resultado.

## Regra prática por classe

| Classe | Tipo | Risco |
|--------|------|-------|
| 1–34 | **Produtos** | 🚨 **ALTO** — titular pode derrubar anúncios de produtos via BPP |
| 35–45 | **Serviços** | ✅ **BAIXO** — classes de serviço não derrubam anúncios de produtos |

{% hint style="warning" %}
⚠️ **Atenção**: a classe 35 é uma das mais registradas (cobre "comércio, varejo") mas **não** permite derrubar anúncios de produtos via BPP. Uma marca só na classe 35 é de baixo risco para você.
{% endhint %}

## Cenários comuns

### Cenário 1: marca só em classes 35–45
**Risco baixo.** Titular não pode usar BPP. Ainda assim, evite copiar marca registrada.

### Cenário 2: marca em classes 1–34
**Risco alto.** Não use o nome exceto se:
- Você é revendedor oficial do produto
- Você tem autorização por escrito do titular

### Cenário 3: marca "Em análise" ou "Indeferida"
**Risco baixo agora**, mas pode mudar. Confira periodicamente.

### Cenário 4: marca não encontrada
**Risco baixo no INPI**. Verifique também Google, redes sociais e Google Patents — alguém pode ter a marca de fato sem registro formal.

## Dicas

- Sempre faça a busca **antes** de criar um anúncio com marca.
- Busque variações (com acento, sem acento, espaços diferentes).
- Se for escalar um negócio próprio, considere **registrar sua marca** no INPI — protege você e dá poder de BPP para evitar cópias.

## Perguntas frequentes

**P: O Marketfacil registra minha marca no INPI?**
R: Não. O Marketfacil **consulta** a base pública do INPI. Para registrar, entre no site oficial do INPI ou contrate um agente/advogado.

**P: A consulta é oficial?**
R: Sim — os dados vêm direto do sistema do INPI. Mesma informação que você veria no site oficial, mais rápida e organizada.

**P: Posso usar uma marca "Em análise"?**
R: Tecnicamente sim, mas o risco é alto: se for concedida depois, o titular pode pedir a retirada dos seus anúncios antigos.
