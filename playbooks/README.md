# Playbooks

As features do Marketfacil são mais poderosas quando usadas **em conjunto**. Os playbooks abaixo são sequências testadas — cada um resolve uma situação concreta combinando 3 a 5 features.

## Playbooks disponíveis

- 🚀 [**Lançando um produto novo**](lancando-produto-novo.md) — do zero até o primeiro anúncio otimizado
- 🔧 [**Recuperando um anúncio que parou de vender**](recuperando-anuncio.md) — diagnóstico e plano de ação
- 🎯 [**Escolhendo um novo nicho**](escolhendo-nicho.md) — validação antes de investir
- ✍️ [**Otimizando um título**](otimizando-titulo.md) — do título atual ao título ideal
- 📈 [**Preparando uma campanha de Ads**](preparando-campanha-ads.md) — checklist pré-investimento

## Como usar os playbooks

1. Escolha o playbook que mais se parece com sua situação.
2. Siga os passos na ordem — cada passo produz um dado que alimenta o próximo.
3. Ao terminar, você tem **ações concretas** para aplicar no Mercado Livre.

{% hint style="info" %}
💡 Cada playbook referencia features específicas do Marketfacil. Se não sabe o que é alguma, o link te leva à explicação.
{% endhint %}
