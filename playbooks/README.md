# Playbooks

As features do Marketfacil se complementam. Os playbooks abaixo mostram **em que sequência usar as features** para cada situação.

{% hint style="info" %}
💡 **Sobre os playbooks**

Os playbooks são **sugestões de sequência** para combinar features — não protocolos fechados. Cada negócio tem sua realidade (margem, nicho, estratégia, escala) e os critérios de "bom" ou "ruim" dependem desse contexto. Use os playbooks como um roteiro inicial e ajuste ao seu caso.
{% endhint %}

## Playbooks disponíveis

- 🚀 [**Lançando um produto novo**](lancando-produto-novo.md) — sequência do zero ao primeiro anúncio otimizado
- 🔧 [**Recuperando um anúncio que parou de vender**](recuperando-anuncio.md) — diagnóstico estruturado
- 🎯 [**Escolhendo um novo nicho**](escolhendo-nicho.md) — verificações antes de investir
- ✍️ [**Otimizando um título**](otimizando-titulo.md) — do título atual ao título ideal (para anúncios novos ou sem vendas)
- 📈 [**Preparando uma campanha de Ads**](preparando-campanha-ads.md) — checagens antes de investir em Mercado Ads

## Como usar

1. Escolha o playbook que se parece com sua situação.
2. Siga os passos na ordem — cada passo alimenta o próximo.
3. Use seu julgamento sobre quanto tempo ficar em cada passo e quais critérios aplicar.
