# Primeiros passos

Se você está chegando agora no Marketfacil, esta é a sequência recomendada:

1. [Criar sua conta](criar-conta.md)
2. [Conectar seu Mercado Livre](conectar-mercado-livre.md)
3. [Tour da plataforma](tour-da-plataforma.md)
4. [Planos e pagamento](planos-e-pagamento.md)

Depois disso, escolha por onde começar:

- Quer saber como está seu anúncio? → [Análise de Anúncios](../analise-de-anuncios/README.md)
- Quer encontrar produtos novos pra vender? → [Busca de Catálogos](../busca-de-catalogos/README.md)
- Seu título está travado, concorrentes passando? → [Agente de Palavras-Chave](../palavras-chave/README.md)
- Vai usar uma marca em anúncios? → [Análise de Marca](../analise-de-marca/README.md)
- Escanear toda sua conta pra limpar tags negativas? → [Auditoria de Tags](../auditoria-tags/README.md)
