# Primeiros passos

Se você está chegando agora no Marketfacil, esta é a sequência recomendada:

1. [Criar sua conta](criar-conta.md) — 1 minuto
2. [Conectar seu Mercado Livre](conectar-mercado-livre.md) — 2 minutos
3. [Tour da plataforma](tour-da-plataforma.md) — 5 minutos

Depois disso, escolha por onde começar:

- Quer saber como está seu anúncio? → [Análise de Anúncios](../analise-de-anuncios/README.md)
- Quer encontrar produtos novos pra vender? → [Buscador de Catálogos](../buscador-de-catalogos/README.md)
- Seu título está travado, concorrentes passando? → [Palavras-Chave](../palavras-chave/README.md)
- Vai lançar uma marca própria? → [Busca INPI](../busca-inpi/README.md)
