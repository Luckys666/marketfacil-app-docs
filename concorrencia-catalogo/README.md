# Concorrência de Catálogo

Informe o link de um catálogo e veja uma tabela completa dos vendedores que competem nele. Perfeito para:

- Decidir seu preço de entrada em um catálogo novo
- Descobrir quem está ganhando a Buy Box e por quê
- Entender a estratégia de frete e comissão dos concorrentes

## Como usar

1. No menu, clique em **Concorrência de Catálogo**.
2. Cole o link do catálogo (formato `mercadolivre.com.br/p/MLB12345`).
3. Clique em **Analisar catálogo**.
4. Aguarde o carregamento — o app busca dados de cada competidor em paralelo.

> 📸 *Screenshot pendente: tabela de competidores com colunas de preço, frete, visitas.*

## O que aparece na tabela

| Coluna | O que significa |
|--------|----------------|
| Vendedor | Nome e link do perfil no ML |
| Preço | Preço atual no catálogo |
| Frete | Grátis, pago, tabela |
| Visitas | Estimativa de visitas diárias |
| Comissão ML | % que o ML cobra do vendedor |
| Posição | Lugar na disputa do catálogo |

## Gráficos

Ao clicar em uma linha da tabela, abre um gráfico com a **evolução das visitas** daquele vendedor específico no catálogo nos últimos dias. Útil pra ver quem está ganhando tração.

> 💡 Vendedor com visitas crescendo + preço agressivo geralmente é quem vai dominar o catálogo nas próximas semanas.

## Perguntas frequentes

**P: O que é Buy Box?**
R: É quem fica em destaque no catálogo — geralmente quem tem a melhor combinação de preço, frete e reputação. Explicado no [Glossário](../GLOSSARIO.md).

**P: Por que alguns dados de competidor aparecem vazios?**
R: O Mercado Livre bloqueia parte das informações de terceiros. O Marketfacil mostra o que é público.

**P: Posso baixar a tabela em Excel?**
R: Essa funcionalidade está planejada — por enquanto, copie os dados manualmente.
